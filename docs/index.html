Why this kit

Copy. Paste. Build better.

oh-my-agent replaces vague instructions with structured prompts that reduce hallucination, surface hidden constraints, and produce consistent output.

📋

Structured Prompts

Every workflow has clear discovery questions, decision points, and output templates. No more "build me an app" — your agent knows exactly what to ask and produce.

🧠

Decision Ledger

Every choice is recorded with rationale and alternatives. No more "why did the agent choose X?" — you get a traceable reasoning trail for every output.

♻️

Reusable Templates

From PRD to deployment plan, every workflow produces structured Markdown documents. Use them as-is or adapt — consistent format across every project.

End-to-end lifecycle

From idea to deployment

Each phase has a dedicated workflow. Start from PRD and move step-by-step. Conditional phases (Design, Debugging) can be skipped when not needed.

Start here. Define product scope, users, constraints. 1 PRD Define
System design, trade-offs, decision records. 2 Architecture Plan
UI/UX design. Conditional — skip for API-only. 3 Design* Plan
Threat model, auth, data protection. 4 Security Plan
Test strategy, coverage, risk assessment. 5 Testing Plan
Break work into ordered tasks. 6 Task List Plan
Agent implements task by task. 7 Build Build
Review code, check quality, approve merge. 8 Review Verify
Release, deploy, monitor. 9 Deploy Ship

* Conditional phase — skip when not applicable. Hover for details. Click node to see workflow.
